The purpose of building this USV

My father wants to measure the depth of the water on the sea，so he and his empolyees built the USV with a sonar. The main controler of the USV is a hardware called Pixhawk.

Pixhawk

Pixhawk is a hardware developed by Swiss University of Technology. It is basically a small computer. pixhawk itself has basic software that can receive gps, sonar, gyroscope, ….., pixhawk has also developed a very powerful platform called open source , So that the pixhawk applications written by players and users all over the world can be uploaded for everyone to download and use.

    • Specifications

    1. Processor
      • 32-bit ARM Cortex M4 core with FPU
      • 168 Mhz/256 KB RAM/2 MB Flash
      • 32-bit failsafe co-processor
    2. Sensors
      • MPU6000 as main accel and gyro
      • ST Micro 16-bit gyroscope
      • ST Micro 14-bit accelerometer/compass (magnetometer)
      • MEAS barometer
    3. Power
      • Ideal diode controller with automatic failover
      • Servo rail high-power (7 V) and high-current ready
      • All peripheral outputs over-current protected, all inputs ESD protected
    4. Interfaces
      • 5x UART serial ports, 1 high-power capable, 2 with HW flow control
      • Spektrum DSM/DSM2/DSM-X Satellite input
      • Futaba S.BUS input (output not yet implemented)
      • PPM sum signal
      • RSSI (PWM or voltage) input
      • I2C, SPI, 2x CAN, USB
      • 3.3V and 6.6V ADC inputs
    5. Dimensions
      • Weight 38 g (1.3 oz)
      • Width 50 mm (2.0”)
      • Height 15.5 mm (.6”)
      • Length 81.5 mm (3.2”)
